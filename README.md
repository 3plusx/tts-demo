# TTS Text-to-speach Demo

Demo von <a href="https://3plusx.io">3plusx</a>, Audio wurde unter Benutzung von CoquiAI und <a href="https://www.thorsten-voice.de/">Thorsten Voice</a> erstellt. Die Timecodes im JSON-Forma wurden mit Stable-TS erzeugt.

Quellcode des UI auf <a href="https://github.com/3plusx/tts-demo">GitHub</a>, 
basierend auf <a href="https://get.foundation/">Foundation</a>.

Mit einem Auszug aus dem Text <a href="https://www.bpb.de/themen/inklusion-teilhabe/behinderungen/521497/teilhabe-und-inklusion/">Teilhabe und Inklusion</a> von Dieter Kulke. Lizenz: CC BY-NC-ND 4.0.
